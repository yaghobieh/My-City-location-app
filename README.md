# My city location application
Welcome to myCity app, You can now store your cities and your place 
you visited. Kind of todo city lost.

Created by React native,
Design by react native stylesheet

## TODO before start:

```
clone https://github.com/yaghobieh/My-City-location-app.git
```

```
npm install
```

```
react-native run-android
Or
react-native run-ios
```

! Don't forget edit in android folder the local.properties file 
To your android APK
